# MEDO_RPG
### Português

**Medo RPG** é um RPG de texto, desenvolvido por mim com o propósito de aprender POO com Python. O jogo em si é bastante rústico (leia-se: bugado) e certamente merece alguns aprimoramentos, mas não me vejo com tempo (ou paciência) de fazer isso. Você pode baixar o [excutável](Exe/Medo-RPG.zip) para Windows ou, e eu recomendo, compilar o jogo você mesmo (nenhuma biblioteca adicional é utilizada além daquelas nativas do Python 3.12.7). O jogo conta com um pequeno tutorial que pode ser acessado com o comando "help".  

### English

**Medo RPG** is a text-based RPG, developed by me with the purpose of learning OOP with Python. The game is pretty rough (read: Buggy) e certainly deserves some improvements, but i don't have the time (or patience) for that. You can download the [executable file](Exe/Medo-RPG.zip) for Windows or, and i recommend, compile the game yourself (no further library is used besides the Python 3.12.7 native ones). The game have a little tutorial that can be acess with the "help" command.
